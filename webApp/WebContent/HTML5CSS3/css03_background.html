background 속성

  • background-color : 배경색 단색설정 : red, blue, rgb코드
  • background-image : 배경을 이미지로 설정 : url(이미지, 파일명);
  • background-repeat : 배경이미지 반복여부 설정 : no-repeat, repeat-x, repeat-y
  • background-position : 배경이미지의 위치 설정 : left, center, right, top, bottom, middle, %
  • background-attachment : 배경이미지 고정 설정 : fixed, scroll
